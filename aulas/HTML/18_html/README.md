## Objetivo da aula
O objetivo desta aula é  entender e praticar sobre como funciona a estrutura do HTML.

-  Tags e atributos
-  Tags de textos
-  Listas
   - Ordenada
   - Não ordenada
   - Aninhadas
-  Rotas e Imagens
-  Hiperlinks
-  Introdução a Tags semânticas
-  Estrutura de sites
-  Elementos de linha e bloco
